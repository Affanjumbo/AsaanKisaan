# Asaan Kisaan Prototype

Demand-aligned agriculture platform prototype covering farmer app, industry dashboard, admin panel, and input/logistics partner module.

## Pages
- `index.html`
- `farmer/index.html`
- `industry/index.html`
- `partners/index.html`
- `admin/index.html`

## Features Included
- Full feature lists for all roles
- Clickable flows and in-browser interactions
- Admin panel with functional tables, filters, and status animations
- Responsive layout with mobile side drawer navigation

## Run
Open `index.html` in a browser.
